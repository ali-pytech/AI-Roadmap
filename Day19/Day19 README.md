# Day 19 — Advanced Error Handling & Custom Exceptions

Today I learned how to create **custom exceptions** in Python. Instead of only catching built‑in errors, I can now design my own rules and raise exceptions when business logic is violated. This makes my systems safer and more professional.



##  Projects I Built

### Iqama Fee Validation
I created a custom exception to ensure iqama fees are always positive. This prevents invalid records.

### VAT Rate Validation
I validated VAT rates to ensure they are between 0 and 1. This keeps financial calculations correct.

### Housing Rent Validation
I raised an exception if monthly rent is unrealistically low for Riyadh housing. This adds business logic to my system.

### Student Grade Validation
I validated student scores to ensure they are between 0 and 100. This prevents invalid grade entries.

### Shopping Cart Validation
I created a custom exception to ensure product quantities are greater than zero. This makes the cart system reliable.



## 🧠 Reflections
